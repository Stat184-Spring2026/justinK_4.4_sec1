# Reproducible Data Analysis Report

This repository was created for the submission of HW #4.3 QMD file into Github. It contains data visualization, self reflection, and other supporting materials creating a reproducible data analysis report.

## Overview

The purpose of this project is to relate how Quarto and R can be used together to create a clean and organized reproducible data analysis document. Moreover, this repository demonstrates the usage of commits, pull requests, and issues to manage the project effectively.
