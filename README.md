# Reproducible Data Analysis Report

This is a repository for the submission of HW #4.3 QMD file into GitHub. It contains data visualization, self-reflection, and other supporting materials, creating a reproducible data analysis report.

## Overview

The purpose of this project is to relate how Quarto and R can be used together to create a clean and organized reproducible data analysis document. 

### Interesting Insight 

One interesting insight from the project was from the busiest airport analysis, where not every airport remained in the top six busiest airports every year. For example, Seoul - Incheon International Airport started appearing in the data set in 2023, while Beijing Daxing International Airport didn't start appearing until 2020.
